# Chat App

Web Chat App which allows you to chat with another user with Web Sockets.

Demo: URL HERE

# Features

1. Creating User Account
2. Account allows you to:
   - Manage Chat:
     - Create
     - Update
     - Delete
   - Messges:
     - Create 
     - Update
     - Delete
   - Join to Another Chat
   - Send Message to Another User
